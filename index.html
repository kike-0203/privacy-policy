Política de Privacidad

Última actualización: septiembre 2026

Este servicio es un bot automatizado que opera a través de la plataforma WhatsApp, utilizando la API oficial de WhatsApp Cloud proporcionada por Meta Platforms, Inc.

La información y los mensajes enviados por los usuarios mediante WhatsApp se utilizan exclusivamente para procesar y responder mensajes dentro de este servicio automatizado.

No recopilamos información personal con fines comerciales, publicitarios ni de marketing.

No vendemos, compartimos ni transferimos los datos de los usuarios a terceros, salvo cuando sea estrictamente necesario para el funcionamiento del servicio a través de la infraestructura oficial de WhatsApp y Meta.

Los datos intercambiados se conservan únicamente durante el tiempo necesario para proporcionar la respuesta solicitada por el usuario.

Al utilizar este servicio, el usuario acepta esta política de privacidad.
